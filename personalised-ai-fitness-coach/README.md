# Personalised AI Fitness & Nutrition Coach

An AI-powered health and wellness platform that generates adaptive workout plans and personalised meal plans based on user biometrics, fitness goals, and daily performance. The system continuously adapts its recommendations by analysing user progress, recovery, and daily check-ins to provide a personalised fitness experience.

**Project:** AI Tools Capstone  
**Domain:** Health & Wellness  
**Difficulty:** Medium  
**Status:** In Development

## Project Overview

Traditional fitness applications often provide static workout routines and meal plans that fail to adapt to changes in user performance. This project aims to develop an intelligent AI-powered fitness coach capable of generating personalised workout and nutrition plans while dynamically adjusting recommendations based on user progress, recovery, and daily feedback.

The primary objective is to distinguish between temporary setbacks and genuine fitness plateaus, ensuring users receive appropriate modifications instead of generic recommendations.

## Objectives

- Generate personalised workout plans using AI.
- Create customised meal plans based on user goals and dietary preferences.
- Continuously adapt recommendations using daily performance check-ins.
- Detect temporary setbacks and long-term fitness plateaus.
- Modify workout plans automatically in case of injuries.
- Visualise user progress through interactive dashboards.

## Project Workflow

1. Collect user information through a personalised onboarding form.
2. Generate a weekly workout plan using Claude.
3. Generate a customised nutrition plan using ChatGPT.
4. Record daily workout performance and meal adherence.
5. Analyse user progress using the adaptive feedback engine.
6. Modify workout and nutrition plans based on user performance and recovery.
7. Visualise long-term progress using Tableau dashboards.

## Core Challenge

The primary challenge of this project is designing an adaptive feedback loop capable of distinguishing between:

- Temporary low-performance days caused by fatigue, stress, poor sleep or insufficient recovery.
- Genuine performance plateaus that require modifications to the workout programme.
- Injury-related limitations that require safe exercise substitutions while maintaining long-term fitness goals.
